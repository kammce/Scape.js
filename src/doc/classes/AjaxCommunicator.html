Defined in js/ScapeJS/ajax-communicator.js:158

Creates a new HTTP request function with a specific action (key), and function.

Parameters:

  • action String

    name of the action that the server should make.

  • path String

    path to the script you want to use.

  • async=false Boolean

    set request to be asycronous or not.

  • done Function

    A wrapper for the success function of the HTTP Request. This function is in adherence to the SCE Server-Client Data Model protocol v0.1.2014. Can be overridden if a different data coming from server is not from SCE. To keep code uniform, it is not advised to override this function and to keep it undefined.

  • error Function

    Override the default error function. To keep code uniform, it is not advised to override this function and to keep it undefined.

Example:

 var io = new AjaxCommunicator();
 io.addRequest("getData", "path/to/script", false);
 io.request("getData", data, function() {
     // on success
 }, function() {
     // on failure
 })

Defined in js/ScapeJS/ajax-communicator.js:200

Issue an HTTP request

Parameters:

  • action String

    name of the action that the server should make.

  • data String | Array | Object

    data to be sent to server (gets JSON stringified).

  • success Function

    function to run if the server request is successful.

  • failure Function

    function to run if server responses with "FAILURE".

Example:

 var io = new AjaxCommunicator();
 io.addRequest("getData", "path/to/script", false);
 io.request("getData", data, function() {
     // on success
 }, function() {
     // on failure
 });
