Constructor Class

Defined in: js/ScapeJS/constructor.js:1
Module: ScapeJS

Constructor loads/holds the functions that should be run right after html has been loaded into a view (<div>). The purpose of constructor is to use the data acquired from AjaxCommunicator and stored in the Storage class to add context to the page. The HTML is a blank template and Constructor fills it in. This is also the class that should be used to add events to elements that exist on a single page as well.
